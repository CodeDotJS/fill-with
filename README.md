# fill-me

> Makes your console colorful.

## Install

```
$ npm install --global fill-me
```

## Usage

```
$ fill-me -w 118 -l 37

  Usage

    $ fill-me

    	Options:
  		    	-w  Change width when you resize your console.                  [required]
 	   	    	-l  Adjust rainbow's height                                     [required]

Missing required arguments: w, l

	$ fill-me -w 118 -l 37

```
